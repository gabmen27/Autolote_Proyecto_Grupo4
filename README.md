# Sistema de Gestión - Autolote Grupo 4

Este es el proyecto de la clase de Programación para la gestión de inventario, clientes y ventas.

## Tecnologías
* Node.js & Express
* MySQL
* JSON Web Tokens (JWT)

## Configuración
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Crear un archivo `.env` basado en `.env.template`.
4. Importar el archivo SQL de la carpeta `/sql`.
